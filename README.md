# FreshTime - Projet HTML Collaboratif
Bienvenue sur le dépôt du projet FreshTime ! Ce document sert de guide et de mémo pour que toute l'équipe puisse travailler efficacement ensemble.

1. Ajouter un nouveau membre
Va sur GitHub > Settings > Collaborators.

Clique sur Add people et cherche ton ami par son pseudo ou email.

ATTENTION : Ton ami doit accepter l'invitation reçue par mail !

2. La Routine de Travail (Workflow)
Étape A : Récupérer (Avant de coder)
Commande : git pull origin main

Étape B : Travailler
Modifie tes fichiers HTML/CSS normalement dans ton éditeur.

Étape C : Publier (Une fois fini)
Préparer les fichiers : git add .

Enregistrer : git commit -m "Description de ton changement"

Envoyer : git push origin main

3. Commandes utiles
git status : Voir quels fichiers sont modifiés.

git log --oneline : Voir l'historique des changements.

git restore . : Annuler les bêtises non enregistrées.
